# UCNature

Block theme for the UC Natural Reserve System.

## About

`ucnature` is a custom WordPress block theme derived from Frost and adapted for the UC Natural Reserve System website. It includes full-site editing templates, custom block styles, a custom CTA block, and theme patterns tailored for editorial and informational content.

## Requirements

- WordPress 6.2+
- PHP 5.7+
- License: [GPLv2](http://www.gnu.org/licenses/gpl-2.0.html) or later

## Getting Started

1. Set up a local WordPress environment.
2. Place this theme in `/wp-content/themes/`.
3. Activate `UC Nature` from Appearance > Themes.
4. Customize templates, patterns, and global styles in the Site Editor.

## Notes

- Main theme bootstrap: `functions.php`
- Global settings: `theme.json`
- Frontend styles: `style.css`
- Block patterns: `patterns/`
- PHP lint: `sh bin/lint-php`
- PHP_CodeSniffer: `sh bin/phpcs-theme`
